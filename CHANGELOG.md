# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-07-06

### Added

- 为 UI 组件新增 SSR 模式支持，可通过 `ui.ssr.enabled` 开启
- SSR 模式下新增 `NG_ALLOWED_HOSTS` 环境变量配置，用于设置允许访问的主机列表

## [0.1.0] - Initial Release

### Added

- note-api: Web API 服务
- note-ui: Web 前端应用
- 数据库备份功能（可选）
